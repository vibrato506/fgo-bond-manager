# FGO Event Farming Optimizer

ゲーム（Fate/Grand Order）のイベント周回を最適化するLINE Botです。  
スクリーンショットをLINEに送信するだけで、AIが画面を解析し、**最小周回数の周回プラン**を自動計算して返答します。

## ✨ 主な機能

### 1. スクリーンショット画像認識（Gemini API）
ゲーム画面のスクショをLINEに送ると、Gemini APIが画像から素材数・ポイントを自動読み取り。  
手入力なしで現在の状況を即座に反映します。

### 2. マルチフェーズ最適化ソルバー
「どの礼装を何周目で外すか」まで考慮した、フェーズ分割型の最適化アルゴリズム。  
素材礼装が不要になったタイミングでポイント礼装に切り替え、余剰を最小化します。

### 3. テキストコマンドによるリアルタイム更新
LINEチャット上でコマンドを送るだけで、素材数や礼装編成をその場で変更可能。

```
所持 金 1500        # 金素材の所持数を1500に変更
必要 銀 3000        # 銀素材の必要数を3000に変更
礼装 ポイント 進化 2  # ポイント礼装（進化）を2枚に変更
ポイント 目標 3000000 # 目標ポイントを300万に変更
```

## 🚀 セットアップ

### 1. リポジトリをクローン

```bash
git clone https://github.com/vibrato506/fgo-farming-optimizer.git
cd fgo-farming-optimizer
```

### 2. 依存パッケージをインストール

```bash
pip install -r requirements.txt
```

### 3. 環境変数を設定

```bash
cp .env.example .env
# .env を編集して各APIキーを入力
```

### 4. LINE Bot を設定

1. [LINE Developers Console](https://developers.line.biz/) でMessaging APIチャネルを作成
2. Webhook URL に `https://your-domain.com/callback` を設定
3. アクセストークンとチャネルシークレットを `.env` に記入

### 5. サーバーを起動

```bash
cd event1
python app.py
```
