# イベントシミュレーター

## イベントの種類と特徴

実際のイベントは、以下の要素が複数組み合わさって構成されることが一般的です（例：ポイント収集 ＋ アイテム収集など）。

### 1. ボックス型
- **特徴**: アイテム抽選（ボックスガチャ）で大量の育成素材、QP、種火を獲得できる。
- **傾向**: リソースの集めやすさが段違いなため、ユーザーからの人気が非常に高い。

### 2. アイテム収集型
- **特徴**: フリクエで特定のアイテムを回収し、クエストの開放条件を満たしていく。
- **傾向**: どの素材をどれだけ集めるか考える必要があり少々手間なため、攻略情報の活用が推奨される。

### 3. 宝箱イベント
- **特徴**: クエストで金・銀・銅の宝箱（レアリティに対応）を獲得する。
- **傾向**: 1箱につき1個の素材が確定で入手でき、ボックス型以上に素材を集めやすいため、非常に有用なイベント。

### 4. ミッション型
- **特徴**: 指定されたミッションを達成することで、新たなストーリーやクエストが開放される。
- **傾向**: FGOで非常にポピュラーな形式。「ストーリークリア → フリクエ開放 → 周回」のループになることが多い。

### 5. ポイント型
- **特徴**: クエストをクリアして特定のポイントを集めることで進行する。
- **傾向**: イベント礼装を装備して周回するシンプルな構成。ただし、クエスト開放条件が複雑な場合もある。
